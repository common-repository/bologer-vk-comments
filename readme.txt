Bologer VK Comments adds comment widget from VK.com for posts and pages with custom settings.

== Description ==

Bologer VK Comments adds comment widget from VK.com for posts and pages, with possibility to have it together with native WordPress comments or override them completely.
In together with customization options.

Major features in Bologer VK Comments include:

* Completely override native WordPress comments
* Set VK widget together with native WordPress comments. Set position of VK widget (over, under or below the header of the native comments)
* Copy code from VK.com and paste into special field to auto fill most of the settings in widget
* Fully customized settings of the widget directly from admin panel

== Installation ==

Install and activate plugin, get widget code from [VK developers page](https://vk.com/dev/Comments), enter it in the special field and press button to automatically set most of the settings for you.
